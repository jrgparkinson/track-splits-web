About

This is a small app to analyse GPS data from track races and calculate split times at arbitrary intervals. We currently only support GPS data from Strava, but analysis of arbitrary .gpx files should be coming soon.

Troubleshooting and FAQs

Searching for activities doesn't work! - your browser may be stopping the code from running. For example, in Google Chrome check for an icon at the right end of the address bar complaining about scripts from unidentified sources.

Bugs, suggestions and more info

More details of the algorithm used to complete split times, including Python code, can be found on GitHub.

If you encounter any bugs or issues, or have any suggestions, then please post these on GitHub too.
