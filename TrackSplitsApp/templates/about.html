About

{% endblock %} {% block content %}

This is a small app to analyse GPS data from track races and calculate split times at arbitrary intervals. You can either load activities from Strava, or upload your own .gpx file for analysis.

{% endblock %} {% block belowJumbotron %}

Planned improvements

  • Bug fixes to the split calculation algorithm, as some GPS tracks aren't currently interpreted correctly.
  • Better support for steeplechase, where the track is slightly different (especially when the jump is on the outside of the bend). At the moment, steeplechase events are mapped onto a standard 400m track.

Troubleshooting and FAQs

Issue: Searching for activities doesn't work. Possible solution: your browser may be stopping the code from running. For example, in Google Chrome check for an icon at the right end of the address bar complaining about scripts from unidentified sources.

Bugs, suggestions and more info

More details of the algorithm used to complete split times, including Python code, can be found on GitHub.

If you encounter any bugs or issues, or have any suggestions, then please post these on GitHub too.
